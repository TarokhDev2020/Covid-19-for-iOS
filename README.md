# Covid-19-for-iOS
Track and get the latest news about covid-19 pandemic such as how many people recovered or died

## Installation ##
* Git and clone <br/>
```bash
$ cd /Users/user/my_project
$ git clone https://github.com/TarokhDev2020/Covid-19-for-iOS.git
```
